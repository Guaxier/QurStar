公共验证码 (GetCode)该接口用于用户登录时进行人机检查

请求样例： 

javascript

plaintext
“data”:{ 
    "way": “getcode”,
}),


参数说明: 

字段名称

类型

必须

描述

way

string

是 (getcode)

请求标识

返回样例：(成功) 

javascript

plaintext
{ 
   message:"获取验证码成功！"
   success: true
     image: "data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABgAAD//gA+Q1JFQVRPUjogZ2QtanBlZyB2MS4wIC"
     codeid："1234"
}

返回样例：(失败) 

javascript

plaintext
{ 
   message:"获取验证码失败！"
   success: false
}

参数说明: 

字段名称

类型

描述

message

string

返回验证码信息

success

布尔

返回获取验证码结果

image

string

返回验证码图片(base64)

codeid

string

验证码id，校验时须同时携带id和验证码

    "phone”:"11111111111",(可选)

    "code": "xxx",
}),

类型

必须

string

string

string

请求样例： 

javascript

plaintext
“data”:{ 
    "way": “login”,
    "username": "xxx",
    "password": "xxx",
    "codeid": "xxx",
    "code": "xxx",
}),


参数说明: 

字段名称

类型

必须

描述

way

string

是 (login)

请求标识

username

string

是

用户名/邮箱

password

string

是

用户密码

codeid

string

是

验证码id，鉴权使用

code

string

是

验证码，人机检测

返回样例：(成功) 

javascript

plaintext
{ 
   message:"登录成功！"
   success: true
     token: "f98545fa6fb3a191d8ea26c1eee20e41753cf995daa65150f3e10f8a216b72e9769924df94238"
}


返回样例：(失败) 

javascript

plaintext
{ 
   message:"用户名或密码错误！"
   success: false
}

字段名称

类型
